rJS (rapidJS) – Generator Utility

Usage:   rjs (generate|gen) <command:name|command:shorthand> [--<arg:name>|-<arg:shorthand> [<option>]?]*
Example: rjs gen instance --path ./apps

• Options:
  --path, -P          Specify a generation target path other than the current working directory.
  --name, -N          Specify which name to use for the generated template.

→ application (app)   Generate template of a minimal web application project for rJS.

Read more at https://rapidjs.org.
