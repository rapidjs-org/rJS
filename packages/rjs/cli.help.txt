rJS (rapidJS)

Usage:   rjs <command> [<sub-command>]* [--<arg:name>|-<arg:shorthand> [<option>]?]*
Example: rjs build ./private --out ./public

• Global Flags:
  --stacktrace      Always print the full stack trace of errors.

→ build             Perform a build.
  • Subcommands:
    1               Path to private file directory.
  • Flags:
    --serve, -S     Whether to serve the files via HTTP for incremental development.
                    When served, the public directory contains files to serve besides the virtually built ones.
  • Options:
    --public        Path to public file directory, i.e. output destination.
    --serve, -S     Alternative server port (default is 7777).

Read more at https://rapidjs.org.
