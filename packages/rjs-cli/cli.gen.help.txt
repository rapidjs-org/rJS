© rapidJS (rJS) – 'Generate' Utility
Usage:   rjs (generate|gen) <command> [--<arg:name>|-<arg:shorthand> [<option>]?]*
Example: rjs gen instance --path ./apps

→ instance      Generate a template of a minimal web application project with rJS.
--path, -P      Specify a generation target path other than the current working directory.
--name, -N      Specify which name to use for the generated application.

Read more at https://rapidjs.org.
